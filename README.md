# Дашборд рекламных кампаний

Рабочее приложение для отслеживания эффективности рекламных кампаний.

🔗 **Демо:** https://tinkriskiss.github.io/dashboard/

## Функционал

✅ Полный CRUD: создание, просмотр, редактирование, удаление кампаний  
✅ Сохранение данных в localStorage  
✅ Импорт и экспорт данных в CSV  
✅ Поиск по названию в реальном времени  
✅ Фильтрация по статусу (активна/на паузе/завершена)  
✅ Сортировка по бюджету и CTR  
✅ Визуализация данных (графики на Recharts)  
✅ Адаптивная вёрстка (мобильная версия)  
✅ Модальные окна подтверждения  
✅ Страница с инструкцией

## Технологии

- **React** (Vite) — фреймворк
- **Tailwind CSS** — стилизация
- **Recharts** — графики
- **GitHub Pages** — деплой

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some Oxlint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
